이상한 문자 만들기

🌟내가 푼 방법🌟 (1)

split() 메서드를 사용하여 문자열 s를 공백기준으로 잘랐다.
홀수번째 인덱스의 알파벳은 소문자로, 짝수번째 인덱스의 알파벳은 대문자로 바꾸어주는 함수를 만들었다. 배열의 끝에 하나 이상의 요소를 추가하는 push() 메서드를 이용하여 빈배열 안에 호출한 함수의 값을 집어넣도록 했다. 이대로 return 하게 되면 배열이 그대로 나오기 때문에 배열의 모든 요소를 연결해 하나의 문자열로 만들어주는 join() 메서드를 이용하여 return 하도록 했다.

🌟내가 푼 방법🌟 (2)

split() 메서드를 썼으니 2차원 배열이라고 생각하고 중첩 for문을 사용했다. j의 for문이 끝났을 때 단어 사이에 push() 메서드를 사용하여 공백을 추가했다. 모든 for문이 끝났을 때는 마지막 단어 끝에 들어간 공백을 없애기 위하여 마지막 요소를 없애주는 메서드인 pop()을 사용했다. 그리고 마지막은 배열을 문자열로 만들어주는 join 메서드를 return하며 끝나게 문제를 풀었다.
중첩 for문은 시간복잡도면에서 최악이라는 것을 예전에 c언어 배울 때 얼핏 봤던 것 같은데, 답을 제출할 때 1번 답과 2번 답의 문제 채점 속도를 보면서 정말 많은 차이가 있다는 것을 깨달았다. 이렇게 노가다처럼 문제 풀고나면 기분은 좋은데 최선의 방법은 아니었다는 것에 조금 슬펐다.
